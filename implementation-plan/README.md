# Nyaya-Setu Implementation Plan

This folder contains the complete implementation roadmap for the Nyaya-Setu autonomous legal defense system.

## Structure

- **Phase 1:** Foundation & Data Setup (Supabase, Auth, RAG)
- **Phase 2:** Backend & Multi-Agent Logic
- **Phase 3:** Frontend & User Experience
- **Phase 4:** Integration, Testing & Deployment

## Tech Stack

- Frontend: React (Vite) + Tailwind CSS
- Backend: FastAPI (Python)
- Database: Supabase (PostgreSQL + pgvector)
- Auth: Supabase Auth
- AI: Gemini 2.5 Pro
- Document Generation: reportlab/pdfkit

## Quick Start

Follow the phases in order. Each phase builds on the previous one and includes user stories, technical tasks, and acceptance criteria.
